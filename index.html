Patient Diagnosis Tool

Project Description:

This project is a Rust-based symptom checker that allows users to input symptoms and receive possible diagnoses based on predefined medical data. It aims to assist telemedicine facilities and pharmacies in improving patient care and streamlining drug ordering processes.
